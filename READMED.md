# 🧠 SmartVision AI

SmartVision AI is a computer vision web application that performs **Image Classification**, **Object Detection**, and **Live Camera Detection** using deep learning models. Built with **Python, Streamlit, TensorFlow, YOLOv8, OpenCV, NumPy, and Pillow**, the project provides a simple and interactive AI-powered vision experience.

---

## 🚀 Features

- Image Classification using MobileNetV2, ResNet50, EfficientNetB0, and VGG16  
- Object Detection using YOLOv8  
- Real-time Live Camera Detection  
- Clean and user-friendly Streamlit interface  

---

## 🧠 Models Used

- MobileNetV2  
- ResNet50  
- EfficientNetB0  
- VGG16  
- YOLOv8  

---

## 📂 Dataset Classes

airplane, bed, bench, bicycle, bird, bottle, bowl, bus, cake, car, cat, chair, couch, cow, cup, dog, elephant, horse, motorcycle, person, pizza, potted plant, stop sign, traffic light, truck
